# Votes for Theme on Computational Communication Workshop

Web Mining Lab is going to hold a Computational Communication Workshop in September at Nanjing University, jointly with School of Journalism and Communication of Nanjing University and SMP Social Media Processing Committee. The workshop is planning to give lectures to one sub-theme for two days and invites votes for the interested theme. Here is the [Vote Link](<http://zhoulvjun.mikecrm.com/QPib0TU>). List of themes 1\. Computer simulations / agent-based modelling 2\. Scraping data from social media 3\. Preprocessing text data for communication research 4\. Topic modeling of text data 5\. Deep learning for communication research 6\. Time series analysis of social media data 7\. Spatial analysis of social media data 8\. Visualization for communication research 9\. Visual analysis for data journalism 10\. Network analysis for communication research (e.g., longitudinal network analysis with Rsiena) 11\. How to teach network analysis to communication students 12\. How to teach data journalism 13\. How to teach coding to communication students 14\. Complex Network & Information Flow 15\. Community Detection 16\. Tie formation and Online Social Network 17\. Machine Learning 18\. Opinion Formation 19\. Programing with Python
