# Collaborator Visits Web Mining Lab

[![](https://webmininglab.github.io/assets/uploads/2012/02/Jonathan-and-Lin-Wu.jpg)](<https://webmininglab.github.io/assets/uploads/2012/02/Jonathan-and-Lin-Wu.jpg>)Lin Wu, a collaborator of Web Mining Lab, visited the Web Mining Lab and delivered tutorials on Python and Hadoop from 21 to 23 Feb.[![](https://webmininglab.github.io/assets/uploads/2012/02/Lin-Wu-lecturing-on-Python.jpg)](<https://webmininglab.github.io/assets/uploads/2012/02/Lin-Wu-lecturing-on-Python.jpg>) There is an increasing need to get data from the Internet and deal with big data for social science research. Lin Wu demonstrated how to crawl user profiles and behavioral data from UGC websites by developing programs using Python and Java. He also introduced the basic ideas and practice of Hadoop, which is a popular framework for reliable, scalable, distributed computing and data storage. Lin Wu received his master degree of computer science from Peking University last year. Web Mining Lab has invited several guest speakers to share their research in various disciplines.
